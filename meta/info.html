This project template is the basis of an expansion board for the TD-XPAH 802.11ah Experimenter's Board.

This base project includes a PCB edge defined as the same size as the TD-XPAH PCB with the connectors placed correctly to align the two boards.

All IO present on the TD-XPAH board is connected to the project through the expansion headers.
